# Understanding Shamir Secret Sharing

Shamir Secret Sharing is a cryptographic technique that allows you to split a secret into multiple shares, requiring a minimum number of shares to reconstruct the original secret.

## The Mathematics Behind It

At its core, Shamir Secret Sharing uses polynomial interpolation. Here's a simplified explanation:

1. Choose a prime number p larger than the secret
2. Generate a random polynomial of degree (t-1)
3. Create shares by evaluating the polynomial at different points

```python
def create_shares(secret, minimum_shares, total_shares, prime):
    coefficients = [secret] + [random.randint(0, prime-1) 
                             for _ in range(minimum_shares-1)]
    
    shares = []
    for i in range(1, total_shares + 1):
        # Evaluate polynomial at point i
        share = evaluate_polynomial(coefficients, i, prime)
        shares.append((i, share))
    
    return shares
```
